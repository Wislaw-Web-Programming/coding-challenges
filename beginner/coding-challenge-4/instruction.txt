Coding Challenge: Create Hyperlinks

Challenge:
Add hyperlinks to your web page. Create links to other web pages or external websites.


1. Inside the <body> section, add the following elements:
   - Create at least two hyperlinks to other web pages or external websites.

2. Use the <a> (anchor) element to create the hyperlinks. Specify the "href" attribute with the URL of the web page or website you want to link to.

3. Optionally, use the "target" attribute set to "_blank" to open the links in a new tab or window.
