Coding Challenge: Create a Simple Web Page

Challenge:
Create a basic HTML web page with a title, heading, paragraph, and an image. Use proper HTML tags for structure.

Instructions:




1. Inside the <body> section, add the following elements:
   - A <h1> element with the text "Welcome to My Web Page."
   - A <p> element with a brief description of your web page.
   - An <img> element for an image. For the source (src) attribute, link to an image located in the "assets" folder of your project. 

3. Save your "index.html" file.

4. Locate an image of your choice and place it in a folder named "assets" within the "simple-web-page" directory.

5. Now, open your "index.html" file in a web browser to see your completed web page.

6. Once you're finished, compare your code to the provided "final" solution in the challenge description to ensure everything matches.
